# CGJteam Lab

CGJteamLab is an open research project dedicated to the formal verification of synthetic Euclidean geometry in Lean 4.

The project develops a reusable geometry library while documenting the complete engineering process of formalization through technical logs, mathematical notes and case studies.

The first case study is Ian Finlay's synthetic proof of the concurrency of the three medians of a triangle.
